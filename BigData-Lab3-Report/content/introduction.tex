\section{Giới thiệu}

\subsection{Bối cảnh bài lab}

Lab 03 của môn Nhập môn Dữ liệu lớn yêu cầu sinh viên giải quyết bốn bài toán phân tích dữ liệu phân tán trên bộ dữ liệu \textbf{Amazon Sale Report}, sử dụng hai nhóm công nghệ:

\begin{itemize}
    \item \textbf{Task 1-1 và Task 1-2:} Advanced MapReduce --- triển khai thuật toán bằng Spark RDD transformations theo tư duy key-value processing.
    \item \textbf{Task 2-1 và Task 2-2:} Spark Structured APIs --- sử dụng DataFrame/Dataset API với Catalyst optimizer.
\end{itemize}

\subsection{Lý do chọn Scala}

Scala được chọn làm ngôn ngữ triển khai cho toàn bộ bốn task vì ba lý do chính:
\begin{enumerate}
    \item \textbf{Full điểm ngôn ngữ:} Theo rubric chấm điểm, sử dụng Scala cho giải pháp hoàn chỉnh được 0.125 điểm/task.
    \item \textbf{Type-safety:} Scala là ngôn ngữ tĩnh kiểu, giúp phát hiện lỗi tại compile-time thay vì runtime --- đặc biệt quan trọng khi làm việc với Dataset API.
    \item \textbf{Native Spark API:} Spark được viết bằng Scala; các API như \texttt{aggregateByKey}, \texttt{reduceByKey}, Dataset encoders hoạt động tự nhiên nhất với Scala, tránh overhead của PySpark serialization.
\end{enumerate}
